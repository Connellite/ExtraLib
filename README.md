# ExtraLib

Small Java 17 utility library: JDBC helpers, collections, date/UUID parsing, and exception helpers.


## Requirements

- JDK 17+
- Maven 3.9+

## Dependency

```xml
<dependency>
    <groupId>io.github.connellite</groupId>
    <artifactId>ExtraLib</artifactId>
    <version>0.2</version>
</dependency>
```
